# Cyber Projects 🛡️

This repository contains my cybersecurity learning projects built using Python.

Currently it includes simple network scanning tools to understand socket programming and network security fundamentals.

---

## Projects

### 1️⃣ Single Thread Port Scanner

File: single-thread-port-scanner.py

Features:
- Basic port scanning
- Uses Python socket programming
- Scans ports sequentially (one by one)

Purpose:
To understand how network ports and socket connections work.

---

### 2️⃣ Multi Thread Port Scanner

File: multi-thread-port-scanner.py

Features:
- Multi-threaded port scanning
- Faster scanning using threading
- Queue-based port distribution
- Saves results to file

Purpose:
To improve scanning performance and understand concurrency in Python.

---

## Technologies Used

- Python
- Socket Programming
- Threading
- Queue

---

## Example Usage

Run the scanner:

python multi-thread-port-scanner.py

Enter:

Target IP  
Start Port  
End Port

Example:

Target IP: 127.0.0.1  
Start Port: 1  
End Port: 1000
